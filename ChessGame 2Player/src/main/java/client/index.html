Game Instructions

1. Each player starts with any Combination of pieces with upto 5 pieces

2. Pieces can move according to their type:

  • Hero1: Moves Forward, Backward, Left, or Right.
  • Hero2: Moves Diagonally (Forward-Left, Forward-Right, Backward-Left, Backward-Right).
  • Pawn: Moves Forward, Backward, Left, or Right.
  • Hero3: Moves in an L-shape (2 steps in one direction and 1 step perpendicular to that direction). Specifically, Hero3 can move in the following directions: Forward-Left, Forward-Right, Backward-Left, Backward-Right, Right-Forward, Right-Backward, Left-Forward, Left-Backward.

3. To initialize the game:

  1. When the board is empty, press the "Initialize Heroes" button.
  2. Player A's pieces can be placed in the 0th row, and Player B's pieces can be placed in the 4th row.
  3. To place a piece, click on a cell in the corresponding row and select the character you want to initialize.
  4. After placing the pieces, press on Start Game to begin.

4. To move a piece, select it and then choose a move option.

5. The game ends when all pieces of one side are eliminated.
